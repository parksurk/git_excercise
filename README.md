# git_excercise

This is repo about git excercise.

## Prerequisites

- python^3.11.5
- FastAPI==0.10

## How to start

```shell
$ git clone {repo addr}
$ cd {repo name}
$ python main.py
```
## Installation

```python
$ brew install python3
$ pip install fastapi==0.10
```

## Features

- 커밋하기
- 커밋 내용 보여주기
- Conventional commit 작성하기
